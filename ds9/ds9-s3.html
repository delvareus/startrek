Time Travel/Hijinks
 
 
 
"There was nothing to indicate that it was today, tomorrow, next week." ... if only we had clocks or watches or literally any time-telling device 🤔
 
 
Watch Recommendation: If you want to watch an okay time-hijinks episode, go for it, but I'm not giving this one a recommendation because the lack of clocks frustrates me. I know it's tough to put clocks on set because it would cause a bunch of continuity issues, but I feel like they could have come up with a better solution here. I know, I know, I'm nitpicking.
